# 🔐 Sistema de Login Simples

Este foi o primeiro projeto que desenvolvi ao voltar a estudar Python.

Antes de começar projetos maiores, decidi criar um sistema de login simples utilizando apenas o que eu ainda lembrava da linguagem. Durante o desenvolvimento precisei relembrar alguns conceitos, corrigir erros e entender melhor a lógica por trás das funções e estruturas de repetição.

Apesar de ser um projeto pequeno, ele representa o reinício da minha jornada na programação.

## 📚 O que foi praticado

- Funções
- Variáveis
- `input()` e `print()`
- Estruturas de repetição (`while`)
- Estruturas condicionais (`if` e `else`)
- Retorno de funções (`return`)

## 🚀 Objetivo

Continuar evoluindo e, conforme for aprendendo novos conceitos, desenvolver projetos cada vez mais completos, como sistemas, automações e aplicações web.
