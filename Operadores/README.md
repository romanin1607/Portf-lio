# ➕ Segunda Aula - Curso em Vídeo

Após concluir a primeira aula do curso, continuei meus estudos de Python aprendendo sobre os **operadores aritméticos**.

Neste repositório está o código desenvolvido durante a **segunda aula** do curso gratuito **Curso em Vídeo**, onde pratiquei cálculos utilizando variáveis, entradas de dados e operações matemáticas.

Durante essa aula aprendi conceitos importantes, como:

* Uso dos operadores aritméticos (`+`, `-`, `*`, `/` e `**`);
* Realização de cálculos com variáveis;
* Entrada de dados utilizando `input()`;
* Conversão de tipos com `int()` e `float()`;
* Formatação de textos utilizando `.format()`.

Para praticar, desenvolvi pequenos programas envolvendo situações do dia a dia, como cálculo de aumento salarial, média de notas, conversão de medidas, descontos e operações com números.

Apesar de serem exercícios simples, eles ajudaram a reforçar a lógica de programação e entender melhor como o Python trabalha com valores e cálculos.

## Objetivo

Registrar meu progresso durante o curso, praticar os conceitos aprendidos e construir uma base sólida em Python para futuramente desenvolver projetos mais completos.
