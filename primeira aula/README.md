# 📚 Primeira Aula - Curso em Vídeo

Antes de voltar a estudar programação de verdade, decidi começar do básico para reforçar os fundamentos.

Neste repositório está o código desenvolvido durante a **primeira aula** do curso gratuito **Curso em Vídeo**, onde estudei os **tipos primitivos** em Python e aprendi conceitos importantes, como:

* Entrada de dados com `input()`;
* Tipos primitivos (`int`, `float`, `bool` e `str`);
* Conversão de tipos (casting);
* Métodos para verificar características de uma string (`isalpha()`, `isdigit()`, `isalnum()`, entre outros).

É um projeto simples, mas representa o início da minha jornada de estudos e servirá para acompanhar minha evolução ao longo do curso.

## Objetivo

Registrar meu progresso, praticar os conceitos aprendidos e construir uma base sólida em Python.
